# Williw Landing Page

A modern landing page for Williw - Decentralized Training Platform.

## Features

- **Hero Section**: Engaging introduction to the Williw platform
- **Features**: Highlighting key capabilities of decentralized training
- **How It Works**: Step-by-step explanation of the platform
- **Download**: Easy access to desktop and mobile applications
- **Multi-language Support**: English and Chinese (中文) languages

## Tech Stack

- React 18
- TypeScript
- Vite
- Tailwind CSS
- i18n (Internationalization)

## Getting Started

### Install Dependencies

```bash
npm install
```

### Development

```bash
npm run dev
```

### Build

```bash
npm run build
```

### Preview Build

```bash
npm run preview
```

## Language

Switch between English and Chinese using the language toggle in the header.
